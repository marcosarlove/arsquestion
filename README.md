## ‚ú® Features

*   **Modern Design:** An interface inspired by the best applications, with smooth animations and an elegant dark theme.
*   **Dynamic Gameplay:** Questions and their answer options are shuffled with each new game, ensuring a unique experience every time.
*   **Multiple Categories:** Choose from various question categories and combine them for a personalized challenge via an interactive modal.
*   **Lifelines:** Use classic lifelines to overcome the toughest questions:
    *   **50:50:** Eliminates two incorrect answers.
    *   **Skip:** Advances to the next question without penalty.
    *   **Hint:** Provides a clue about the current question.
*   **Scoring System:** Accumulate points in the style of "Who Wants to Be a Millionaire?".
*   **Instant Feedback:** Receive immediate visual and auditory feedback for each answer.
*   **Extensible:** Adding new question categories is simple and fast.
*   **Zero Dependencies:** Built without frameworks and without the need for `npm`. Pure JavaScript!
